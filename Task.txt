URL Shortener:
Design a URL Shortener service that takes in a valid URL and returns a shortenedd URL,
redirecting the user to the previously provided URL.
Also, keep track of total visits/clicks on the URL.

Routes
POST/URL - Generates a new short URL and returns the shortened URL in the format example.com/random-id.
GET/URL/:id - Redirects the user to original URL
GET/URL/analytics/:id - Returns the clicks for provided short id.
